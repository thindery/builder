How do I..?

Agile Toolkit delivers simplicity on top of very powerful code-base. Many things you can think of are probably have been already implemented on Agile Toolkit

If you have any question on how to do something with Agile Toolkit, ask it on Stack Overflow with atk4 as a tag.

Updates

Your software will continue to be compatible with 4.0.x releases of Agile Toolkit. With minor changes you can also upgrade to 4.1.x. You can get newer version from here: http://agiletoolkit.org/download.

When doing upgrade, replace your current "atk4" directory with the newer release.

Online documentation

Documentation located on http://agiletoolkit.org/doc is aimed at 4.1 branch of Agile Toolkit. Refer to http://agiletoolkit.org/whatsnew for the differences between versions.

Stay in touch with framework releases and major upgrades by signing up to our http://agiletoolkit.org/newsletter.
